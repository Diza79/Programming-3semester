Реализовать линейный, ветвящийся и циклический алгоритмы, 
используя 32-х разрядную систему команд процессора 80х86. 
Программы реализуются в среде Microsoft Visual Studio с 
использованием ассемблерной вставки. Линейный алгоритм 
реализуется одним способом, ветвящийся алгоритм реализуется 
одним способом, циклический алгоритм реализуется тремя способами. 
Таким образом, в отчете должно быть представлено 5 небольших 
программ.

Пример реализации программы: 

// test_con.cpp: главный файл проекта.

#include "stdafx.h"
#include "stdio.h"

using namespace System;

int main() {
    int a,b,c,d;
    char s;

    printf("\n\rInput A,B,C");
    scanf("%d %d %d",&a,&b,&c);
    _asm{
                mov eax,[a]
                mov ebx,[b]
                mov ecx,[c]
                add eax,ebx
                imul ecx
                mov [d],eax
    }
    printf("\n\r Result: %d",d);
    scanf("%1c",&s);

    return 0;
}
