Лабораторная работа № 3

Связные списки. Библиотека стандартных шаблонов (STL)

Задание
Разработать и отладить программу на языке C++, реализующую работу со списком двумя способами: используя библиотеку стандартных шаблонов (STL) и самому реализовать структуру данных -  связный список. Выполнить оценку временной и емкостной сложности программы.
 

Варианты
1. Написать функцию, которая по списку L строит два новых списка: L1– из положительных элементов и L2 – из отрицательных элементов списка L.
2. Определить, является ли список упорядоченным по возрастанию.
3. Сформировать список целых чисел, вводимых пользователем, в том порядке, в котором вводятся эти числа, но без повторений элементов.
4. Написать функцию, которая оставляет в списке L только первые вхождения одинаковых элементов.
5. Определить количество различных элементов списка действительных чисел, если известно, что его элементы образуют возрастающую последовательность.
6. Имеется список целых чисел. Продублировать в нем все четные числа.
7. Написать функцию, которая по двум данным линейным спискам формирует новый список, состоящий из элементов, одновременно входящих в оба данных списка.
8. Написать функцию, которая по двум линейным спискам L1 и L2 формирует новый список L, состоящий из элементов, входящих в L1, но не входящих в L2. 
9. Написать функцию, которая в линейном списке из каждой группы подряд идущих одинаковых элементов оставляет только один.
10. Написать функцию, которая удаляет из списка элементы, входящие в него только один раз.
