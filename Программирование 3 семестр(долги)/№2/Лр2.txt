Описание

1) По теме вашего курсового проекта (вы должны были уже сообщить мне ее название), 
придумать и описать несколько основных типов объектов, которые будут полезны 
при реализации вашего курсового проекта и которые разумно представлять в виде 
структур языка C (не менее 8 шт. + они должны быть взаимосвязаны).

2) Выделить набор значимых свойств этих объектов и разработать набор 
соответствующих структур языка C для их представления в программе.

3) Разработать набор функций на языке C для работы с этими структурами.

4) В main продемонстрировать работу со “статическими” и динамическими 
переменными этих типов. 

5) По проделанной работе оформить отчет.

ПРИМЕЧАНИЕ: При выполнении данной лабораторной работы не используйте 
объектно-ориентированные конструкции языка С++.
