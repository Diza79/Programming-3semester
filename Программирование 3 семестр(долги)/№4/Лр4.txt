Описание

1) Для задания из лабораторной работы №3 
реализовать необходимые классы на языке Java;

2) Продемонстрировать их использование в консольном 
Java-приложении

3) При выполнении задания разрешается использовать 
IDE только для написания кода программы. Для сборки, 
упаковки и запуска необходимо использовать  только JDK/JRE; 

4) Продемонстрировать запуск вашего приложения на 
альтернативной платформе (смартфон, Linux и пр.).
